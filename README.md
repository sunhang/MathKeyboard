# 数学公式键盘
该app旨在让用户在手机上更高效的输入数字公式和数学符号，以便用户能够在互联网上交流数学知识。同时，该app将会加入物理辅助教学等功能，会通过动画的方式来描述物理世界的运动规律（比如自由落体、惯性、摩擦阻力等）
